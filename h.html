Debian安装Docker稳定版（官方最新）

一、前置准备――更新一下系统的各种软件包

    apt update

    apt full-upgrade

二、安装Docker（最新版Docker已经包含docker compose）

  1. 移除旧的软件包

    2. apt-get remove docker docker-engine docker.io containerd runc

  2. 下载需要的依赖项

    3. apt-get install ca-certificates curl gnupg lsb-release

  3. 创建一个目录用来存docker仓库的GPG密钥

    4. mkdir -p /etc/apt/keyrings

  4. 下载docker仓库的GPG密钥

    5. curl -fsSL https://download.docker.com/linux/debian/gpg | sudo gpg --dearmor -o /etc/apt/keyrings/docker.gpg

  5. 将docker仓库添加到系统的软件源列表中

    6. echo "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.gpg] https://download.docker.com/linux/debian $(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null

  6. 刷新系统的软件源列表

    7. apt-get update

  7. 安装docker

    8. apt-get install docker-ce docker-ce-cli containerd.io docker-compose-plugin
